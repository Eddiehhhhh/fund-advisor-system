# 🎯 智能基金理财顾问系统

一个多维度基金分析系统，每日自动分析市场，识别处于历史低点的指数基金，生成投资建议并推送。

## 系统架构

```
┌─────────────────────────────────────────────────────────────┐

│  • 移动平均线        • 管理费率         • 北向资金              │
```

## 核心功能

### 1. 智能低点识别
- **历史分位点分析**: 识别PE/PB处于历史5%分位点以下的基金
- **技术面共振**: 结合RSI超卖、MACD金叉、价格跌破布林带下轨等信号
- **趋势预判**: 分析近期资金流向，识别即将反转的赛道

### 2. 多维度评分体系
```
总评分 = 估值评分(30%) + 技术评分(25%) + 资金评分(20%) + 基本面评分(15%) + 情绪评分(10%)
```

### 3. 每日报告内容
- 📊 市场概览 (大盘指数表现)
- 🎯 推荐买入 (评分>80分的基金，附理由)
- 📈 关注列表 (评分60-80分，等待更好时机)
- ⚠️ 风险提示 (近期跌幅较大的赛道)
- 💡 操作建议 (仓位配置建议)

## 配置

### 方式一：本地运行

1. 创建 `.env` 文件（已创建，包含敏感信息，已加入.gitignore）
2. 确认 `.env` 中包含：
   - `DEEPSEEK_API_KEY`: DeepSeek API密钥
   - `EMAIL_AUTH_CODE`: QQ邮箱SMTP授权码
3. 运行：`python main.py`

### 方式二：GitHub Actions 自动运行（推荐）

1. 将代码推送到GitHub仓库
2. 设置GitHub Secrets：
   ```bash
   ./setup_github_secrets.sh <你的GitHub用户名> <仓库名>
   ```
   或手动设置：
   - `DEEPSEEK_API_KEY`: `sk-493c343144244a54819701156d737503`
   - `EMAIL_AUTH_CODE`: 你的QQ邮箱授权码
3. 每天上午9:00自动收到邮件报告

## 运行

```bash
# 手动运行一次
python main.py

# 启动本地定时任务 (每日9:00自动分析)
python main.py --schedule

# GitHub Actions手动触发
# 进入仓库 → Actions → 每日基金理财日报 → Run workflow
```
